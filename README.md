# Vacation Rental Management System 🏡

A backend project built using Node.js, Express, and MongoDB, designed to simulate a vacation rental publishing platform.

The system manages users (publishers), apartments, categories, and cities, with full authentication and authorization support.

---

## 🚀 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- REST API
- bcrypt (password encryption)

---

## 🧩 Key Features

### 👤 Users (Publishers)
- User registration
- User login
- JWT-based authentication
- Secure password hashing

### 🏠 Apartments
- Create apartment listings
- Update listings
- Delete listings
- Filter apartments by:
  - City
  - Category
  - Price
  - Number of beds
  - Publisher

### 🏙 Cities & Categories
- Add new cities / categories
- Retrieve full lists

---

## 🔐 Security & Authorization
- JWT-based Authentication & Authorization
- Protected routes for sensitive actions
- Ownership validation before update/delete operations

---

## 📦 Project Structure
- Controllers – business logic
- Models – data schemas
- Routes – API endpoints
- Middleware – authentication & authorization

---
